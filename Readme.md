# Task 5: CSS Hero Section (Display, Position, Float)

**Author:** D P Rithvik Kumar

## Description
This is my submission for the Laundry Services Hero Section task. The goal of this assignment was to build a responsive, split-screen layout without relying on Flexbox. 

To solve this, I used `display: inline-block` to align the navbar items and the left/right hero sections. I also utilized viewport units (`vw` and `vh`) to ensure the design fits perfectly on the screen without triggering a scrollbar. 

It was a bit tricky getting the inline-block elements to align without wrapping to the next line! I learned that if you set widths exactly to 50%, browser spacing can push the second element down, so I adjusted the widths to 45% and used `vertical-align: top` to keep everything balanced. 

## How to Run
1. Download or clone this repository to your local machine.
2. Ensure that both `index.html` and `style.css` are located in the same folder.
3. Double-click the `index.html` file to open it directly in your web browser (Chrome, Edge, Safari, etc.).
4. Alternatively, you can view the live project by clicking the deployed GitHub Pages link on the submission portal.
